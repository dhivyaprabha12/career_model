# Career Bot 🤖💬

Career Bot is an AI-powered chatbot designed to provide **emotional support, career guidance, and personalized roadmaps** for users who feel uncertain about their future.

Unlike traditional bots, this project focuses on combining **motivation + practical direction**, helping users not only choose a path but also feel confident about it.

---

## 🌟 Overview

Many students and beginners struggle with confusion, self-doubt, and lack of direction when it comes to careers.  
Career Bot aims to bridge this gap by acting as a **supportive AI companion** that listens, understands, and guides.

---

## ✨ Key Features

- 💬 Emotion-aware conversational responses  
- 🧠 AI-generated career suggestions  
- 🛤️ Step-by-step roadmaps for chosen paths  
- 💡 Motivational and confidence-building responses  
- 🌐 Simple and interactive web interface  

---

## 🛠️ Tech Stack

- **Backend:** Python, Flask  
- **AI/NLP:** Hugging Face Transformers, PyTorch  
- **Other:** Tokenizers, JSON configurations
